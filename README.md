# 🛕 DarshanEase

DarshanEase is a temple darshan ticket booking web application that allows devotees to reserve slots and book tickets for darshan at various temples online — eliminating the need for long physical queues.

## 🌟 About the Project

DarshanEase simplifies the spiritual journey by offering a convenient, technology-driven platform where devotees can:
- Browse available darshan time slots
- Select specific pooja types
- Book tickets seamlessly from home or on the go
- Plan visits in advance with flexible scheduling

This project modernizes the traditional temple visit experience, replacing manual queuing and on-site ticket purchases with a fast, user-friendly digital booking system.

## 🚀 Features

- User-friendly slot browsing and booking interface
- Pooja type selection
- Real-time booking management
- Secure backend with RESTful APIs
- Responsive UI built with React.js

## 🛠️ Tech Stack

**Frontend:** React.js, JavaScript, CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Tools:** Git & GitHub, Postman, VS Code

## ⚙️ System Requirements

**Software:**
- Node.js v16+
- npm v8+
- MongoDB
- Modern browser (Chrome/Firefox)

**Hardware (Recommended):**
- Intel i5 (8th Gen) / AMD Ryzen 5 or better
- 8 GB RAM minimum (16 GB recommended)
- 1 GB free storage

## 📦 Installation & Setup

1. Clone the repository
```bash
   git clone https://github.com/Rajesh1234-rajesh/DarshanEase.git
   cd DarshanEase
```

2. Install frontend dependencies
```bash
   npm install
```

3. Run the development server
```bash
   npm run dev
```

4. (If backend exists separately) Navigate to backend folder, install dependencies, and start the server
```bash
   npm install
   npm start
```
